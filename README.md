# useful-RegEx
Collection of the useful RexExs


# Index


# Contents
* Software version
```regex
(?:(\d+)\.)?(?:(\d+)\.)?(?:(\d+)\.\d+)
```

* IPv4
```regex
^(([0-9]|[1-9][0-9]|1[0-9]{2}|2[0-4][0-9]|25[0-5])\.){3}([0-9]|[1-9][0-9]|1[0-9]{2}|2[0-4][0-9]|25[0-5])$
```

* Domain
```regex
^([a-zA-Z0-9]([a-zA-Z0-9\-]{0,61}[a-zA-Z0-9])?\.)+[a-zA-Z]{2,6}$
```
